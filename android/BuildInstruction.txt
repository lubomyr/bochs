#!/bin/sh

. setenv-android.sh

Now via Android SDK Manager (for call sdk manager via console type android). We need download latest version:
Android SDK Tools
Android SDK Platform-tools
Android SDL Build tools
Android 6.0 (API 23)
+SDK Platform (API 23)
